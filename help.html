📘 Grade Moderator – User Guide

🔰 What is this system?

This software helps instructors:

  • Upload student marks
  • Automatically align them with ERP data
  • Visualize grade distributions
  • Assign and adjust grades interactively
  • Export final results for submission

🧭 Overall Workflow (Simple)

  1. Go to New Grades
  2. Upload or paste marks
  3. Upload ERP file
  4. View histogram and assign grades
  5. Export final results

🏠 Home Page

This is the starting page with three options:

  • New Grades → Standard workflow
  • New Grades (Cleaner) → Recommended (easier)
  • View Grades → Load previously saved work

🆕 Creating Grades (Recommended Method)

Step 1: Enter Details

  • Subject Name
  • Maximum Marks

Step 2: Paste Marks

  • Copy from Excel/Google Sheets
  • Paste directly
  • Format: Column 1 = ID, Column 2 = Marks

Step 3: Select Starting Row

Usually 1 or 2 depending on header rows.

Step 4: Upload ERP CSV

This file is used to match students automatically.

Step 5: Click "Show Histogram"

📊 Histogram & Grading Page

This is the most important screen.

What you see:

  • Marks distribution graph
  • Grade cutoffs
  • Student counts
  • Average marks & MGPV

What you can do:

  • Adjust grade boundaries using slider
  • Edit grades manually
  • Search individual students
  • View analytics (pie/bar charts)

Special Features:

  • Search Grades → find specific students
  • Grade Analytics → visualize distribution
  • Edit Mode → manually change grades

💾 Saving & Exporting

  • Save ERP CSV → final file for upload
  • Save Grades → saves work as JSON
  • Print PDF → report for records

📂 Viewing Existing Grades

If you saved your work earlier:

  • Upload the JSON file
  • The system restores everything

📑 Sorting & Export Page

You can organize results by:

  • Campus
  • Branch
  • Both

Then export clean tables for reporting.

⚠️ Important Notes

  • Ensure Student IDs are correct
  • Marks must be numeric
  • ERP file format should not be modified
  • Special grades (W, I, etc.) are handled automatically

💡 Tips for First-Time Users

  • Use the Cleaner version (paste-based)
  • Start with a small dataset to test
  • Always save JSON backup
